# VOIDIA Project

VOIDIA es una plataforma modular de inteligencia artificial open‑source para producción audiovisual rápida y ética. 
Este repositorio contendrá los módulos iniciales de música, voces y animación, junto con documentación y ejemplos.

Estado actual: prototipo funcional en desarrollo (2026).
# VOIDIA Project

## Español
VOIDIA es una plataforma modular de inteligencia artificial open‑source para producción audiovisual rápida y ética.  
Este repositorio contendrá los módulos iniciales de música, voces y animación, junto con documentación y ejemplos.  

Estado actual: prototipo funcional en desarrollo (2026).  
Roadmap: MVP en 2 meses, modularidad audiovisual en meses 3–5, marketplace en meses 6–9.  

## English
VOIDIA is an open‑source modular AI platform for fast and ethical audiovisual production.  
This repository will host the initial modules for music, voices, and animation, along with documentation and examples.  

Current status: functional prototype in development (2026).  
Roadmap: MVP in 2 months, audiovisual modularity in months 3–5, marketplace in months 6–9.
